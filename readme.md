# Buscador de Perfil GitHub
 
Aplicação web que permite buscar perfis do GitHub pelo nome de usuário, exibindo avatar, nome e bio do perfil encontrado.
 
## Funcionalidades
 
- Busca de perfil pelo nome de usuário via [GitHub API](https://docs.github.com/en/rest/users/users)
- Exibição de avatar, nome e bio do usuário
- Feedback visual de loading durante a busca
- Mensagem de erro para usuários não encontrados
 
## Tecnologias
 
- HTML5
- CSS3
- JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com/) v3 (CDN)
- [GitHub REST API](https://docs.github.com/en/rest)
